# Welcome to Pyoway

**Pyoway** is a personal website and knowledge base built entirely with modern Rust tooling.

This documentation site contains blog posts, tutorials, and reference articles covering:

- **Rust** — Language guides, patterns, and framework notes
- **Web Development** — WASM, modern frontend, and server-side topics
- **Tools & Workflow** — Cargo tooling, CI/CD, and development environment setup
